# My Website

Vite + React + Tailwind v4 專案，含 Framer Motion、Lucide React、clsx、tailwind-variants。

## 本地執行

```bash
# 安裝依賴
npm install

# 開發
npm run dev

# 建構（請確保無誤）
npm run build

# 預覽建構結果
npm run preview
```

## 技術棧

- **Vite** — 建構工具
- **React** — UI
- **Tailwind CSS v4** — `@import "tailwindcss"`（無 base/components/utilities）
- **Framer Motion** — 動效
- **Lucide React** — 圖標
- **clsx / tailwind-variants** — 樣式工具

## 目錄約定

- 靜態資源：`public/`
- 源碼：`src/`
- 別名：`@/*` → `src/*`（見 `jsconfig.json`）
